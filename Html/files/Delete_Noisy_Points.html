Subroutine Delete_Noisy_Points (Pat, NoisyP, FileInfo)


Type(Diffraction_Pattern_Type)

Intent(in out)

Pat

Pattern

Integer

Intent(out)

NoisyP

Number of noisy points found

Logical, optional

Intent(in)

FileInfo

If .true. then create a file containing information about the noisy points


Delete noisy points in a Pattern.


The procedure actual consider that a point is a noisy point when:

    1. Point "i" is non monotonic
    2. Dif1 < .or. Dif2 < sc


where:

sc= 8.0*sqrt( (ci2+ci1+cd1+cd2)/4.0)

Dif1=cc - 2.0*ci1+ci2

Dif2=cc -2.0*cd1+cd2


and


ci2:        counts at         left-left position

ci1:        counts at              left position

cc:        counts at        current position

cd1:        counts at            right position

cd2:        counts at       right-right position
