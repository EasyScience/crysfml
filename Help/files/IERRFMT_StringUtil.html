IERR_FMT

INTEGER :: IERR_FMT


This variable contains information about the error on FINDFMT procedure.


The error code is according to next information:



Code

Meaning

-2

FORTRAN read error

-1

End of file

0

No Error

1

Empty format descriptor (0 field)

2

Data string read error

3

Integer field found real

4

Begged dot, sign or "e" character

5

Invalid character in an integer field

6

Invalid field in format descriptor

7

Invalid character in a numeric field

8

0 character in current field

9

Format string length exceeded

10

Separator missing

11

Incomplete E or D format

12

Incomplete number
