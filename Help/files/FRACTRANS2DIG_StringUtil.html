FRAC_TRANS_2DIG

FRAC_TRANS_2DIG (V, CHARF)


REAL, DIMENSION(3)

INTENT (IN)

V

Input vector

CHARACTER (LEN=*)

INTENT (OUT)

CHARF

String



Subroutine returning a string describing a 3D translation vector written in fractional form as quotient of 2-digit integers with sign.


Example:


V= (0.3, -0.4, -5.5)  -> CHARF="(3/10,-2/5,-11/2)"
