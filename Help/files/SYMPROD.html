SYM_PROD

FUNCTION SYM_PROD(SYMA, SYMB, MODLAT)


TYPE(SYM_OPER_TYPE)

INTENT(IN)

SYMA

Space group

TYPE(SYM_OPER_TYPE)

INTENT(IN)

SYMB

Space group

LOGICAL, OPTIONAL

INTENT(IN)

MODLAT


Obtain the symmetry operation corresponding to the product of  two operators. The return is a variable of type SYM_OPER_TYPE

If MODLAT=.true. or it is not present, the translation part of the resulting operator is reduced to have components < 1.0
