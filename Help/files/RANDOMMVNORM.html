RANDOM_MVNORM

RANDOM_MVNORM (N, H, D, F, FIRST, X, IER)


INTEGER

INTENT (IN)

N

Number of variates in vector (input,integer >= 1)

REAL, DIMENSION (N)

INTENT (IN)

H

Vector of means

REAL, DIMENSION (N*(N+1)/2

INTENT (IN)

D

Variance matrix (j> = i)

REAL, DIMENSION (N*(N+1)/2

INTENT (OUT)

F

Parameter of distribution (0 < real)

LOGICAL

INTENT (IN)

FIRST

.TRUE. if this is the first call of the routine or if the distribution has changed since the last call of the routine.

.FALSE. otherwise

REAL, DIMENSION (N)

INTENT (OUT)

X

Delivered vector

INTEGER

INTENT (OUT)

IER

= 1 if the input covariance matrix is not +ve definite

= 0 otherwise



Subroutine that generates an n variate random normal vector using a Cholesky decomposition.


NOTE: Reference to the book from Dagpunar, J. Principles of random variate generation, Clarendon Press, Oxford, 1988.
