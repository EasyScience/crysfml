GET_STABILIZER

SUBROUTINE GET_STABILIZER (X, SPG, ORDER, PTR)


REAL(KIND=CP), DIMENSION(3)

INTENT(IN)

X


Position vector

TYPE(SPACE_GROUP_TYPE)

INTENT(IN)

SPG


Space group

INTEGER

INTENT(OUT)

ORDER


Number of sym.op. keeping invariant the position x

INTEGER, DIMENSION(:)

INTENT(OUT)

PTR


Array pointing to the symmetry operators numbers of the stabilizer (point group) of x


Subroutine to obtain the list of symmetry operator of a space group that leaves invariant an atomic position. This subroutine provides a pointer to the symmetry operators of the site point group.
